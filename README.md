# Bellezza Tech

Лендинг с корзиной для компании Bellezza Tech — производителя обеденных столов из спеченного камня.

## Особенности

- **13 моделей столов** с настраиваемыми параметрами
- **19 текстур камня** на выбор
- **3 уровня исполнения**: Standard, Premium, Ultimate
- **B2B режим** с оптовыми ценами
- **Корзина** с оформлением заказа
- **Интеграция Firebase** — сохранение заказов в Realtime Database
- **Telegram уведомления** — мгновенная отправка заказов в Telegram
- **Адаптивный дизайн** — работает на всех устройствах

## Технологии

- HTML5, CSS3, JavaScript (vanilla)
- Firebase Realtime Database
- Telegram Bot API
- GitHub Pages (хостинг)
